# expensetracker

A modern, responsive, and mobile-ready Expense Tracking application built with a "Mobile-First" approach. This app allows users to manage their daily finances, visualize spending summaries, and install the site directly onto their phones as a Progressive Web App (PWA).

## ✨ Key Features
* **PWA Integration:** Installable on Android and iOS with a custom home screen icon and splash screen.
* **Offline Functionality:** Uses Service Workers to cache assets, allowing the app to load without an internet connection.
* **Responsive Design:** Fully adaptive UI that works seamlessly on mobile, tablet, and desktop viewports.
* **Glassmorphism UI:** Modern design aesthetics using CSS backdrop filters and translucent layers.
* **Secure Authentication:** User login and registration flows including a secure password generator.
* **Interactive Dashboard:** Real-time balance updates and spending visualizations.

## 📱 How to Install
### On Android (Chrome)
1. Open the website.
2. Tap the "Add to Home Screen" prompt or find it in the browser menu.
3. The app will now appear in your app drawer.

### On iOS (Safari)
1. Open the website.
2. Tap the **Share** button (square with arrow).
3. Scroll down and select **Add to Home Screen**.
